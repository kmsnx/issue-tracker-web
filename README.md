# Issue Tracker

Simple issue tracker. Add your issues and change theirs' states.


### Development
* Install
    ```
    npm ci --legacy-peer-deps
    ```

* Start
    ```
    npm run start
    ```

For more scripts check out package.json

### Deployment
* Build
    ```
    npm run build
    ```

* Deploy
    ```
    npm run deploy
    ```
